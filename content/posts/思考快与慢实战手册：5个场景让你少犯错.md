> 知道那么多道理，还是过不好这一生？问题可能在于：你知道，但没在关键时刻用起来。

《思考，快与慢》讲了上百种认知偏差，但读完就忘是常态。这篇不聊理论，直接给你**5个高频场景 + 具体检查清单**，下次遇到类似情况，掏出手机对照着看就行。

---

## 场景一：超市购物——别让"锚"钩住你的钱包

**你是不是也这样**：

看到"原价199，现价99"，瞬间觉得捡了大便宜，二话不说就扔购物车。"买二送一"的促销明明只需要一个，最后提了三盒回家。导购先说"这款高端型号要5000"，再说"这款入门款只要2000"，你心想2000真便宜，完全忘了自己本来只想花500。

**为啥会这样？**

第一个进入你大脑的数字会成为"锚"，后续判断都围着它转。199让99显得便宜，5000让2000显得划算——哪怕2000本身根本不值那个价。你的急性子被锚钩住了，慢性子还在打盹。

**下次购物前，打开这个清单看一眼**：

- [ ] 这个"原价"是真的，还是商家随便标的？
- [ ] 如果没有任何促销，我还会买吗？
- [ ] 算单价而不是总价，大包装不一定更划算
- [ ] 先查一下电商平台的日常价格，别被"限时特价"忽悠
- [ ] 如果没有"买X送X"，我真的需要这么多吗？

**一个小技巧**：逛超市前列好清单，设定预算上限。看到促销时，先走开5分钟，让急性子冷静下来，再决定要不要回去买。往往5分钟后，你就不想要了。

## 场景二：股票/基金投资——别让"可得性"替你决策

**你是不是也这样**：

刷到某只股票连续涨停的新闻，心跳加速，手指忍不住往交易软件上滑。朋友推荐了一只"必涨"的基金，你立马跟投，连它是干啥的都没搞清楚。某行业最近负面新闻多，你觉得整个行业都不行了，赶紧抛售。

**为啥会这样？**

最近发生的事、媒体报道多的信息，更容易被大脑提取，你就误以为它们更普遍、更重要。连续涨停不代表会继续涨，负面新闻不代表行业完蛋。你的急性子被"容易想起"的信息绑架了。

**下单前，把这5个问题看三遍**：

- [ ] 我买它是因为"最近听说"，还是因为真的研究过？
- [ ] 如果这只股票跌了30%，我能接受吗？（这是在测试你的损失厌恶）
- [ ] 查一下这个行业的长期数据，而不是最近一周的新闻
- [ ] 朋友的推荐，他自己投了多少？赚了还是亏了？
- [ ] 如果这是别人的钱，我还会这么决策吗？

**一个小技巧**：设立"冷静期规则"——任何投资决策，至少隔一个晚上再执行。把想买的理由写下来，第二天再看一遍，往往会有不同想法。急性子热得快凉得也快，给它一晚时间。

## 场景三：求职/跳槽——别让"怕失去"困住你

**你是不是也这样**：

现在的工作很痛苦，但想到要换环境就焦虑，觉得"至少现在稳定"。拿到新offer，总盯着"要失去的东西"——熟悉的同事、常去的餐馆、攒了三年的人脉——而不是"能得到的东西"。甚至拒绝了一个薪资更高但title稍低的机会，因为"降职太没面子"。

**为啥会这样？**

人对损失的敏感度是收益的两倍。换工作意味着"失去现有的一切"，这种恐惧往往压倒了"新机会的收益"。你的急性子被"怕失去"的情绪淹没了，完全没算过账。

**纠结去留时，问自己这几个问题**：

- [ ] 如果现在这份工作是我刚拿到的offer，我还会接受吗？（这是打破现状偏见的绝招）
- [ ] 把"留下"和"离开"的利弊写在纸上，量化打分
- [ ] 3年后回头看，哪个选择更可能让我后悔？
- [ ] 新工作的"不好"是确定的，还是我想象出来的？
- [ ] 问3个信任的朋友：如果是你，会怎么选？

**一个小技巧**：用"外部视角"——想象这是你的朋友面临的选择，你会给他什么建议？我们给别人建议时往往更理性，因为急性子不会代入自己的情绪。

## 场景四：薪资/价格谈判——谁先开口谁占便宜

**你是不是也这样**：

HR问"你的期望薪资是多少"，你报了一个数，后来才发现要少了。卖二手东西，买家先出价，你的心理预期就被拉低了。买房时中介先带你看几套超预算的，再看目标价位的，突然觉得"好便宜"。

**为啥会这样？**

谈判中先出价的一方设定"锚点"，对方的心理预期会围绕这个锚点调整。哪怕这个锚点很离谱，也比没有锚点强。你的急性子急着回答，慢性子没来得及提醒你：先开口的人，往往吃亏。

**谈判前，先把这5个问题想清楚**：

- [ ] 我查了行业薪资数据/市场价格区间吗？（建立自己的锚）
- [ ] 我的底线是多少？低于什么数字我会直接拒绝？
- [ ] 如果对方先出价，我有没有准备好说"这个数低于我的预期"？
- [ ] 除了钱，还有什么可以谈的？（假期、title、远程办公等）
- [ ] 如果谈崩了，我的Plan B是什么？

**一个小技巧**：尽量让对方先出价。如果必须你先开口，给一个比你实际期望高20-30%的数字，留出被砍价的空间。记住：锚点一旦设定，就很难摆脱了。

## 场景五：项目规划——别让"这次不一样"坑了你

**你是不是也这样**：

预估项目3个月完成，结果拖了半年。觉得自己能每天早起跑步，坚持了一周就放弃。新年计划列了10条，年底一条都没实现，但明年还是信心满满地列10条。

**为啥会这样？**

我们总觉得自己比别人强、这次会不一样，于是低估难度、高估自己的执行力。每个项目都以为"这次会顺利"，结果每次都延期。急性子过度自信，慢性子懒得提醒。

**做计划时，先过一遍这个清单**：

- [ ] 类似项目平均耗时多久？（查历史数据，用外部视角）
- [ ] 如果最乐观是3个月，最悲观可能是多久？
- [ ] 我把计划告诉别人了吗？（公开承诺能增加执行力）
- [ ] 计划中有没有预留缓冲时间？（建议加50%）
- [ ] 如果只能实现一半，我会选哪一半？

**一个小技巧**：用"预演失败法"——假设项目已经失败了，倒推可能的原因。这种"事前验尸"能帮你的慢性子发现盲点，避免急性子的盲目乐观。

## 把清单存进手机

这5个场景覆盖了生活中大部分重要决策。建议你把对应的检查清单存进手机备忘录，遇到相关情况时掏出来对照。

**说到底，核心原则就三条**：

**慢下来** —— 别让急性子急着拍板，给它5分钟、一晚、甚至一天的时间。

**外部视角** —— 想象这是别人的事，你会怎么建议？往往能看得更清楚。

**写下来** —— 把想法落到纸上，比空想靠谱得多。纸不会骗你，但脑子会。

认知偏差不可能完全消除，但可以被管理。每次少犯一点错，长期积累下来就是巨大的差距。

> 急性子负责反应，慢性子负责把关。关键是，别让急性子独自做决定。
